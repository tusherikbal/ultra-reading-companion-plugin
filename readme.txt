== Description ==

Ultra Reading Companion is an all-in-one lightweight plugin designed to enhance the user reading experience on your WordPress posts. It provides a visual reading progress bar, an estimated reading time indicator, and a sleek, customizable scroll-to-top button.

By giving readers a clear idea of how long a post will take to read and showing their progress visually, you can significantly reduce bounce rates and improve engagement.

=== Key Features ===
* **Reading Progress Bar:** A smooth bar at the top of the page that fills as the user scrolls.
* **Estimated Reading Time:** Automatically calculates and displays the reading time for your posts.
* **Scroll to Top Button:** A customizable button with multiple Dashicons support to help users navigate back to the top easily.
* **Fully Customizable:** Change colors, bar height, padding, and margins directly from the settings page.
* **Lightweight & Fast:** Built with clean code and minimal assets to ensure your site stays fast.
* **Secure:** Follows WordPress.org security standards with full data sanitization and escaping.

== Installation ==

1. Upload the `ultra-reading-companion` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'Reading Tool' menu in your WordPress dashboard to configure the settings.

== Frequently Asked Questions ==

= Does it work on pages? =
Currently, it is optimized to work on single posts, which is where reading metrics are most valuable.

= Can I change the progress bar color? =
Yes, you can easily change the bar color, height, and reading time background from the plugin settings page.
